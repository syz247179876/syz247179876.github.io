Linux上安装python的虚拟环境

Linux上安装python的虚拟环境

我的之前的一篇笔记已经介绍了window上安装python的虚拟环境，这篇来介绍下Linux上如何安装python的虚拟环境。

安装步骤

① 更新下pip

pip install upgrade pip

② 安装virtualenv

pip install virtualenv

③ 创建虚拟环境

virtualenv myvenv

注：默认情况下, 虚拟环境中不包括系统的site-packages，如果要使用系统的包

virtualenv --system-site-packages myvenv

④ 激活进入虚拟环境

我的activate在虚拟环境下的bin目录下

因此执行source MyBlog/bin/activate就可以激活虚拟环境

⑤ 退出虚拟环境

deactivate
